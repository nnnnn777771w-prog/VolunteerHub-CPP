# 🛡️ Volunteer Hub — Система координации штаба ЧС

Интеллектуальная консольная система на языке C++ с цветным GUI-интерфейсом. Программа создана для автоматизации процессов распределения ресурсов, учета волонтеров и мгновенной обработки экстренных заявок от населения в условиях чрезвычайных ситуаций.

---

## 📋 Описание функционала

В приложении реализован комплекс модулей для полноценного управления кризисным штабом:

1. Двухуровневая авторизация (Администратор/Гость): 
   - Координатор (Администратор) — полный доступ к базе данных (регистрация карточек, начисление часов, экспорт отчетов).
   - Гость — режим ограниченного просмотра аналитики и таблиц. Функции изменения данных визуально заблокированы.
2. Учет и smart-валидация данных: Регистрация волонтеров и заявок с жесткой фильтрацией категорий (Медицина, Поиск, Еда, Ремонт) и автоматической проверкой форматов (ФИО через нижнее подчеркивание _).
3. Экстренный мэтчинг (Алгоритм поиска): Автоматическое посимвольное сопоставление навыков спасателей с категориями происшествия в реальном времени.
4. Приоритет спасения (Сортировка Bubble Sort): Сортировка заявок по убыванию уровня критичности (от 1 до 3). Критические заявки уровня ЧС 3 автоматически поднимаются наверх таблицы и подсвечиваются ярко-красным цветом.
5. Аналитический модуль: Расчет процента выполнения задач штаба и суммарного вклада работы волонтеров в человеко-часах.
6. Энергонезависимая БД: Полноценное сохранение и автозагрузка данных через файловые потоки (requests.txt, volunteers.txt).
7. Официальный экспорт: Выгрузка сводного отчета координатора в текстовый файл report.txt.

## 🛠️ Инструкция по запуску

### Системные требования:
* Компилятор с поддержкой стандарта C++11 или выше (GCC, Clang, MSVC).
* Операционная система: Windows 10 / 11 (для корректной работы цветного WinAPI интерфейса).

### Пошаговый запуск:
1. Склонируйте репозиторий или скачайте исходный код:
   git clone https://github.com/ваш_ник/volunteer-hub.git
2. Откройте проект в вашей среде разработки (Visual Studio, VS Code, CLion) или скомпилируйте через терминал:
   g++ main.cpp -o volunteer_hub
3. Запустите скомпилированный файл volunteer_hub.exe.

### ⚠️ Решение проблем с отображением русского языка (Кодировка)
Если при первом запуске в консоли вместо кириллицы появились иероглифы («кракозябры»), вы можете настроить параметры под свой компьютер. Откройте main.cpp, найдите начало функции int main() и добавьте нужный вариант:

* Для старых консолей Windows (Стандарт):
  SetConsoleCP(1251);
  SetConsoleOutputCP(1251);
  setlocale(LC_ALL, "Russian");
* Для современных терминалов / VS Code (UTF-8):
  SetConsoleCP(65001);
  SetConsoleOutputCP(65001);
  setlocale(LC_ALL, "Russian");

## 🚀 Пример использования

1. Вход в систему: При старте программа предложит выбрать роль. Выберите 1 (Координатор) и введите логин admin и пароль 1234.
2. Регистрация: Перейдите в пункт меню 1 и создайте волонтера (например, Petrov_Petr` с навыком Поиск) и экстренную заявку (например, Потерялся_человек_в_лесу с категорией Поиск и уровнем ЧС 3).
3. Обработка: Перейдите в пункт меню 4. Вы увидите, что созданная заявка автоматически поднялась в самый верх таблицы и подсветилась красным.
4. Мэтчинг: Выберите пункт 2 (Экстренный мэтчинг), введите ID заявки, и система мгновенно выдаст вам карточку волонтера Petrov_Petr как идеального кандидата для выполнения задачи.
