# Программа Volunteer Hub для координации волонтеров

Это консольное приложение на C++, которое помогает вести учет волонтеров и заявок о помощи во время чрезвычайных ситуаций. Программа заменяет бумажные журналы и позволяет быстро находить нужных людей для решения задач.

## Что умеет программа

1. Вход по ролям: можно войти как Администратор (логин admin, пароль 1234), чтобы добавлять и менять данные, или как Гость — только для просмотра списков.
2. Учет данных: программа хранит анкеты волонтеров (ФИО, навыки, группа крови) и список заявок (описание проблемы, уровень срочности).
3. Сортировка по важности: самые срочные заявки автоматически поднимаются на самый верх списка и подсвечиваются в консоли красным цветом.
4. Автоматический поиск: функция поиска (мэтчинг) сама подбирает волонтеров, у которых навыки совпадают с темой заявки.
5. Защита от вылетов: если вместо цифры случайно ввести букву, программа не сломается, а попросит повторить ввод.
6. Работа с файлами: все данные сохраняются в файлы volunteers.txt и requests.txt, поэтому при перезапуске ничего не теряется. Также можно выгрузить отчет в файл report.txt.

## Как запустить

1. Открыть файл main.cpp в любой программе для разработки (Visual Studio, VS Code, CLion).
2. Скомпилировать и запустить проект.
* Примечание: настройки кодировки (setlocale / SetConsoleCP) удалены из кода для поддержки разных операционных систем. Настройте кодировку терминала (UTF-8 или CP1251) в параметрах вашей среды разработки, если текст отображается некорректно.

## Пример использования

1. Запустить программу, выбрать пункт 1 (Координатор) и ввести пароль.
2. Добавить волонтера с навыком "Поиск" и добавить заявку с темой "Поиск".
3. Открыть список заявок — задача со срочностью 3 будет в самом верху.
4. Включить поиск волонтеров по ID заявки, и система сама покажет созданного волонтера.
