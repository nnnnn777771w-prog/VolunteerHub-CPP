# VolunteerHub — Система координации волонтеров при ЧС

Консольная программа на C++ для автоматизации работы волонтерского штаба.

## 📋 Что умеет программа (Функционал):
1. **Регистрация:** Добавление волонтеров и заявок на помощь от населения.
2. **Экстренный мэтчинг:** Автоматический поиск волонтеров по совпадению их навыков с категорией заявки.
3. **Учет часов:** Закрытие заявок и автоматическое начисление рабочих часов волонтерам.
4. **Сортировка:** Вывод списка заявок, отсортированных по уровню критичности (пузырьковый алгоритм).
5. **Аналитика:** Расчет эффективности работы штаба (% закрытых задач и общие человеко-часы).
6. **Отчеты:** Экспорт сводной таблицы со всеми заявками в файл `report.txt`.

## 🛠️ Особенности реализации:
* Данные сохраняются и загружаются автоматически через файлы `requests.txt` и `volunteers.txt`.
* Написана защита от некорректного ввода (программа не ломается при вводе букв вместо чисел).
* Встроена фильтрация мусора (нельзя ввести отрицательные часы или несуществующую группу крови).
* Использованы только базовые библиотеки (`<iostream>`, `<fstream>`, `<windows.h>`). 
* Вместо динамических контейнеров использованы структуры и статические массивы.

## 💻 Как запустить:
1. Скомпилировать файл `main.cpp` в любой IDE (Visual Studio или другое).
2. Запустить. Русский язык в консоли настроен автоматически.
