# 🛡️ Volunteer Hub — Система координации штаба ЧС

Интеллектуальная консольная система на языке C++ для автоматизации процессов распределения ресурсов, учета волонтеров и обработки экстренных заявок от населения в условиях чрезвычайных ситуаций.

---

## 🔑 Обновления: Система авторизации и ролей

В систему внедрено разграничение прав доступа для обеспечения безопасности персональных данных волонтеров и защиты базы от некорректных изменений.

### Доступные режимы:
1. **Координатор (Администратор)** — полный доступ к управлению штабом (регистрация, экстренный мэтчинг, начисление часов, экспорт отчетов).
   - **Логин:** `admin`
   - **Пароль:** `1234`
2. **Гость** — режим ограниченного просмотра для внешних наблюдателей. Доступен только просмотр отсортированных таблиц ЧС, аналитики эффективности и встроенной инструкции. Функции изменения базы данных заблокированы.

---

## 🚀 Ключевые возможности и алгоритмы

* **Экстренный мэтчинг (Алгоритм поиска):** Автоматическое посимвольное сопоставление категорий происшествия с навыками свободных спасателей в реальном времени.
* **Приоритет спасения жизней (Сортировка):** Реализация классического алгоритма пузырьковой сортировки (`Bubble Sort`) по убыванию уровня критичности задач. Самые экстренные вызовы всегда находятся на вершине таблицы.
* **Smart-Валидация (Защита от дурака):** - Потоки ввода защищены от буквенного мусора (`cin.clear()`, `cin.ignore()`).
  - Строгий фильтр на текстовые категории (разрешены строго: *Медицина, Поиск, Еда, Ремонт*).
  - Проверка формата ФИО волонтеров (обязательное использование нижнего подчеркивания `_`).
* **Аналитический модуль:** Расчет процента выполнения задач штаба и суммарного вклада работы волонтеров в человеко-часах.
* **Энергонезависимая БД:** Полноценное сохранение и автозагрузка данных через файловые потоки `ifstream` / `ofstream` (`requests.txt`, `volunteers.txt`).
