# Volunteer Hub — Система координации штаба ЧС

## Описание проекта
Volunteer Hub — это консольное приложение на C++, созданное для автоматизации работы оперативного штаба в условиях чрезвычайных ситуаций. Программа помогает координаторам вести учет волонтеров, оперативно принимать заявки о помощи от населения и автоматически подбирать нужных специалистов под конкретные задачи.
---

## Описание функционала
Программа поддерживает два режима работы: Координатор (Администратор) и Гость (Только просмотр).

* **Разграничение прав доступа:** Авторизация администратора по логину и паролю. Гостевой режим блокирует функции изменения баз данных.
* **Учет ресурсов штаба:** Регистрация волонтеров (с указанием ФИО, группы крови, навыков) и добавление входящих заявок на помощь.
* **Валидация данных:** Встроенная проверка корректности ввода для предотвращения сбоев при некорректном типе данных. Ограничение значений (группы крови 1–4, уровни ЧС 1–3, строго фиксированные категории навыков).
* **Алгоритм экстренного мэтчинга:** Автоматический поиск и вывод списка волонтеров, чья специализация совпадает с категорией конкретной заявки.
* **Приоритизация задач:** Реализация ручной пузырьковой сортировки (Bubble Sort) для вывода наиболее критических заявок уровня ЧС в топ списка.
* **Модуль аналитики:** Расчет процентного соотношения закрытых задач к общему числу заявок и подсчет суммарно отработанных волонтерами часов.
* **Файловая подсистема:** Автоматическое сохранение баз данных в файлы (requests.txt, volunteers.txt) при завершении сессии, а также экспорт сводного отчета в файл report.txt.

## Инструкция по запуску
Проект разработан и протестирован в среде разработки Microsoft Visual Studio.

1. Склонируйте репозиторий или скопируйте исходный код файла `main.cpp`.
2. Создайте пустой проект C++ (Консольное приложение) в Visual Studio.
3. Добавьте файл `main.cpp` в каталог исходных файлов проекта.
4. Нажмите сочетание клавиш **Ctrl + F5** (или выберите "Локальный отладчик Windows") для компиляции и запуска приложения.

---

## Пример использования
1. **Авторизация:** При запуске выберите пункт меню `1`. Введите логин `admin` и пароль `1234` для получения прав Координатора.
2. **Регистрация волонтера:** Перейдите в пункт меню `1`, затем выберите `1`. Введите ФИО через нижнее подчеркивание (например, `Ivanov_Ivan`), укажите навык `Медицина` и группу крови `2`.
3. **Создание заявки:** В том же меню выберите пункт `2`. Введите описание проблемы `Нужна_аптечка`, категорию `Медицина`, уровень критичности `3` (ЧС) и требуемое время работы `2` часа.
4. **Запуск мэтчинга:** Выберите в главном меню пункт `2` и введите числовой ID созданной заявки. Система выведет таблицу с волонтером `Ivanov_Ivan`, так как его специализация соответствует запросу.
5. **Сохранение и выход:** Выберите пункт меню `0`. Программа запишет текущее состояние массивов в текстовые файлы на диске и завершит процесс.
